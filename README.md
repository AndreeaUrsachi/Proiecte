  # Galentine's Day Card
Acest proiect contine o felicitare interactivă pentru Galentine's Day. Este un proiect mic care folosește HTML, CSS și JavaScript pentru a crea un card cu un mesaj și 2 butoane. Dacă butonul de "Da" este apăsat atunci felicitarea dispare si apare un videoclip de mulțumire și mai multe gif-uri drăguțe.

  # Funcționalități 
- felicitare interactivă pentru Galentine's Day
- animații drăguțe și design plăcut
